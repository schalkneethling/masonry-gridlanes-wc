Mixed wall · Live controls

Masonry playground

A live playground built on the mixed wall demo. Toggle custom-element attributes on and off, adjust their values, and see how they affect the layout in real time.

min-column-width is the primary layout control here. Row mode is experimental and intentionally excluded from this playground for now.

Applied attributes

gap="16" min-column-width="250"

Controls

Enable an attribute, then tune it. Disabled controls remove that attribute entirely.

This uses the same mixed content as the existing demo while turning the custom element into a live inspector. It is meant to make attribute behavior easier to learn, compare, and tune.
