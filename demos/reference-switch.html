Reference switcher

Column and row masonry

A proving-ground demo based on a compact reference layout: the same nine cards switch between fixed-column masonry and fixed-row masonry so the difference in packing direction is easy to see.

Columns are fixed, rows pack tight. Items flow into the shortest column, eliminating the gaps you would get with a regular grid.

min-column-width="320" mode="columns"

This demo intentionally forces the JavaScript fallback so it can act as a stable reference for the library's own row-mode behavior.
